# Kotlin-react-template

## Usage:
1. Clone repository
2. Edit settings.gradle.kts to change project name
3. Edit build.gradle.kts to change group and version
4. Edit src/main/resources/index.html to change title and script name
5. Open src/main/kotlin/Index.kt and put your app there

## To add more pages:
1. Copy index.html, change name, title and root div id (keep root-)
2. Create new class for the page
3. Add mapping to Main.kt (without root-)
